Django 入门

Python的WEB框架有Django、Tornado、Flask 等多种，Django相较与其他WEB框架其优势为：大而全，框架本身集成了ORM、模型绑定、模板引擎、缓存、Session等诸多功能。

基本配置

创建Django程序

  • 创建项目: django-admin startproject sitename
  • 启动程序: python manage.py runserver 0.0.0.0:80

    启动程序并且监听在0.0.0.0的80端口，即：主机上的80端口，通过访问主机上的任意IP:80端口都可以访问到程序

  • 创建App: python manage.py startapp app01

    创建一个APP，创建如下文件

  • python manage.py makemigrations

    会根据setting.py下配置的INSTALL_APPS下的所有app，生成对应得SQL文件保存在migrations目录下(只生成不存在的)

  • python manage.py migrate

    会根据migrations目录下的脚本创建对应的数据库表到数据库

  • python manage.py createsuperuser

    创建Django管理后台的超级用户的用户名和密码
